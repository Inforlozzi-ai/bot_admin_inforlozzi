# bot_admin_inforlozzi

UserBot Telegram Pro com IA (OpenAI) + Bot Admin Revendedor — by Inforlozzi AI

## Instalação rápida

```bash
curl -fsSL https://raw.githubusercontent.com/Inforlozzi-ai/bot_admin_inforlozzi/main/install.sh \
  -o install.sh && chmod +x install.sh && bash install.sh
```

## Estrutura

```
bot_admin_inforlozzi/
├── bot.py              ← UserBot com menu IA
├── install.sh          ← Instalador / Gerenciador
├── requirements.txt
├── .env.example
└── bot-admin/
    ├── admin_bot.py    ← Bot Admin Revendedor
    ├── requirements.txt
    └── .env.example
```
